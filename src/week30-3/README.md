# 30週目ミニドリル 3問目

## 問題

学生名、月毎ごとの学習時間を算出してください

`select students.name 学生名, ? 学習月, sum(studies.hours) 月ごとの学習時間 from students join studies on studies.student_id = students.id group by 学習月, name;`
上記sqlの?部分に適切な処理を入れてください
